TECH STACK

FoxVault is built entirely on Google Cloud Platform. Every layer — compute, storage, encryption, auth, and observability — uses a managed GCP service with no self-hosted infrastructure.

⚡
Cloud Run Functions
Serverless compute for upload, download, and list. Python 3.11, per-request billing, VPC-isolated.
Python 3.11
☁
Cloud Storage
Durable encrypted PDF storage. Uniform IAM, 7-year retention, versioning, zero public access.
CMEK · Uniform IAM
🔑
Cloud KMS
Customer-managed AES-256 encryption. 365-day auto-rotation, per-key service account binding.
AES-256-GCM
🌐
Cloud API Gateway
Managed HTTPS entry with JWT validation. All three function endpoints secured behind the gateway.
JWT Auth
🔒
Identity Platform
Firebase-compatible email/password auth. Issues JWTs for all protected API Gateway routes.
Email / Password
📋
Cloud Audit Logs
Data Read and Write on Cloud Storage and KMS. Every access and decryption logged automatically.
Data R/W Enabled
📊
BigQuery
Long-term compliance archive via Log Router sink. Queryable 7-year audit trail.
Log Sink
📡
Pub/Sub
Security alerting bus. Unauthorized access attempts trigger alerts via Cloud Monitoring.
Security Alerts
🛡
Serverless VPC Access
Private connector for Cloud Run → GCP APIs over Google's internal network only.
Private Egress
